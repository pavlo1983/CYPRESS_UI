
Windows installation

Clone provided repository from github;
Run 'npm install cypress --save-dev' (install Cypress to your project);
Run 'npx cypress open' (from the project folder);

Mac installation

Clone provided repository from github;
Get into a project folder
Run command 'npx'
Run command 'cypress install'
Run command 'node_modules/.bin/cypress open'



Delete demo tests from cypress/integration folder (all except file ibiza.specs.js!!!) (optional Windows and Mac)
